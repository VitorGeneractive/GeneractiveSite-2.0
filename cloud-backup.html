FAQ – Cloud Backup

Perguntas Frequentes sobre Proteção de Dados

1) O que diferencia o Cloud Backup corporativo do backup local tradicional?

O Cloud Backup oferece redundância geográfica, automação e segurança criptografada, enquanto o backup local depende de infraestrutura física e está sujeito a falhas humanas ou desastres.

2) Como garantir que os dados estão realmente protegidos?

A Generactive utiliza criptografia de 256 bits, backups imutáveis e auditorias automáticas, além de monitoramento contínuo para garantir a integridade das cópias.

3) É possível restaurar apenas um arquivo específico?

Sim. O Cloud Backup permite recuperação granular de arquivos, versões e bancos de dados individuais, sem necessidade de restaurar o volume completo.

4) O serviço está em conformidade com a LGPD?

Sim. Aplicamos controles de acesso, registro de logs e armazenamento seguro em data centers compatíveis com LGPD e GDPR.
